# Markdown Kullanım Rehberi
Bu proje, Markdown işaretleme dilinin temel sözdizimini ve kullanım örneklerini içerir. Markdown, metinleri kolayca biçimlendirmek ve dokümantasyon oluşturmak için hafif bir işaretleme dilidir.

## İçerik
- Başlıklar
- Kalın ve İtalik Yazılar
- Listeleme
- Bağlantılar ve Görseller
- Kod Blokları

Markdown dilini öğrenmek isteyen herkes için yararlı bir referans olabilir.

## Kullanım
Markdown dosyasını incelemek için [markdown-dosyası.md](./markdown.md) bağlantısına tıklayın.
